perf简介

perf 功能非常强大， 号称瑞士军刀， 对性能很多问题， 都可以尝试用perf 来帮忙解决， 它能够进行函数级与指令级的热点查找。它由一个叫“Performance counters“的内核子系统实现，基于事件采样原理，以性能事件为基础，支持针对处理器相关性能指标与操作系统相关性能指标的性能剖析，可用于性能瓶颈的查找与热点代码的定位。
